# CodeAlpha_Hangman_game
This project is a simple command-line Hangman Game developed as part of the CodeAlpha Python Development Internship. The player tries to guess a randomly selected word by entering one letter at a time. The game tracks used letters, remaining attempts, and updates the word display after each guess.

Features

Command-line interface

Random word selection

Tracks correct and incorrect guesses

Prevents repeated inputs

Beginner-friendly Python logic

Requirements

Python 3.13

Optional: a words.txt file containing word lists

How to Run

Install Python 3.13.

Download or clone the project.

Navigate to the project folder.

Run the following command:

python hangman.py

How to Play

Guess one letter at a time.

Correct letters reveal their position in the word.

Incorrect letters reduce the number of attempts.

Win by guessing the full word before attempts run out.
